Resumo Financeiro

A funcionalidade de "Resumo Financeiro", é responsável por fornecer diversos valores financeiros importantes da aplicação de forma resumida. Por exemplo, é possível saber o saldo Disponível e o saldo Bloqueado.

Os formatos monetários serão sempres strings contendo como separadores de milhar o "." (ponto), e como separador de casas decimais a "," (vírgula). E também terão geralmente 2 casas decimais.

Saldo Disponível

Requisição

/resumoFinanceiro/saldoDisponivel?tipo=cartao

O caminho acima é relativo. Consulte o caminho absoluto em Introdução

Tipo: GET
Body: Vazio

Cabeçalhos (HTTP)

Content-Type application/json

Parâmetros (URL)

NomeFormatoValores PossíveisObservação
tipoTextocartao, boleto

Resposta

{
    "data": "170.286,08",
    "responseStatus": [
        {
            "status": "ok",
            "mensagem": "Sucesso!"
        }
    ]
}

Resumo Financeiro (Completo)

Requisição

/resumoFinanceiro/get?tipo=cartao

O caminho acima é relativo. Consulte o caminho absoluto em Introdução

Tipo: GET
Body: Vazio

Cabeçalhos (HTTP)

Content-Type application/json

Parâmetros (URL)

NomeFormatoValores PossíveisObservação
tipoTextocartao, boleto

Resposta

{
    "data": {
        "codigo": "219",
        "saldoDisponivel": "170.286,08",
        "saldoBloqueado": "584.558,27",
        "valorTransferenciaPendente": null,
        "valorTotalTransferido": null,
        "valorTotalPagoCredito": null,
        "valorTotalPagoDebito": null
    },
    "responseStatus": [
        {
            "status": "ok",
            "mensagem": "Sucesso!"
        }
    ]
}
