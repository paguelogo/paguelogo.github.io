Recuperar Senha

Para utilizar essa funcionalidade, deve ser feito uma requisição do tipo 'POST', contendo os dados de login e/ou e-mail do usuário do qual se deseja recuperar a senha.

Requisição

/recuperarSenha

O caminho acima é relativo. Consulte o caminho absoluto em Introdução

Tipo: POST
Body (JSON):

{
    "login": "[login]",
    "email":"[email]"
}

Cabeçalhos (HTTP)

Content-Type application/json

Atributos

NomeFormatoValores PossíveisObservação
loginTexto Livre (sem espaços em branco)login do usuário do qual deseja recuperar o acesso
emailTexto Livre (sem espaços em branco)e-mail do usuário do qual deseja recuperar o acesso

Só é preciso informar um dos dois atributos. Contudo caso seja informado os dois atributos, não há problema.


Resposta

{
    "data": null,
    "responseStatus": [
        {
            "status": "ok",
            "mensagem": "Os dados de recuperação de senha foram enviados para o e-mail cadastrado para esse usuário. Siga os passos indicados no e-mail para poder recuperar seu acesso ao sistema."
        }
    ]
}

Resposta ilustrativa. Ela serve apenas para informar que o procedimento deu certo.
